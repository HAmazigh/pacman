pacman by schuay

Pacman clone using SDL. Guide Pacman through a maze and collect pac-dots while avoiding ghosts.

Dependencies

sdl sdl_gfx sdl_mixer sdl_ttf

Install

Installation on linux: * dependencies: 'sdl' 'sdl_ttf' 'sdl_gfx' 'sdl_mixer' * run "make" * all done! In case you're running ArchLinux, here's a premade PKGBUILD: http://aur.archlinux.org/packages.php?ID=28014 Installation on windows: * download and extract the precompiled windows version: http://cloud.github.com/downloads/schuay/pacman/pacman_sdl_windows.zip * or compile from source
